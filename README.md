# Digit Recognizer (Machine Learning)

A machine learning project that recognizes handwritten digits (0–9) using image classification techniques in Python.

---

## Features

- Handwritten digit prediction (0–9)
- Image preprocessing and normalization
- Model training and testing
- Accuracy evaluation
- Predicts digits from input images

---

## Technologies Used

- Python
- NumPy
- Pandas
- Scikit-learn / TensorFlow (use whichever you used)
- Matplotlib (if used for visualization)

---

## Project Purpose

The goal of this project was to understand and implement basic machine learning concepts for image classification and apply them to real-world handwritten digit recognition.

---

## How It Works

1. Dataset of handwritten digits is loaded
2. Data is preprocessed and normalized
3. Machine learning model is trained
4. Model is tested for accuracy
5. New images are predicted as digits (0–9)

---

## Future Improvements

- Improve accuracy using deep learning (CNN)
- Deploy as a web application
- Real-time digit recognition from drawing canvas
